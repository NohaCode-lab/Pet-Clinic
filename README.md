# 🐾 Pet-Clinic

**Modern Veterinary Clinic Website**

Pet-Clinic is a responsive and modern frontend web project designed for a veterinary clinic. It provides a clean, structured, and user-friendly interface to showcase clinic services, veterinary doctors, pet care information, and customer testimonials.

This project demonstrates real-world frontend development skills using HTML, CSS, JavaScript, and Bootstrap.

---

## 🚀 Live Demo

https://github.com/NohaCode-lab/Pet-Clinic

---

## 📌 Overview

Pet-Clinic is a static frontend website that simulates a real veterinary clinic platform. It focuses on building a professional UI layout with modern design principles and responsive behavior across all devices.

The project is ideal for portfolio presentation and demonstrates strong frontend structuring skills.

---

## ✨ Features

### 🏥 Services Section
- Displays veterinary clinic services
- Includes vaccination, grooming, nutrition, and pest control
- Icon-based visual design for better UX

### 👨‍⚕️ Doctors Section
- Showcase of veterinary doctors
- Profile cards with images and specializations
- Clean and structured layout

### 🐶 Pet Care Content
- Informational sections for pet owners
- Health tips and educational content

### 💬 Testimonials
- Customer feedback section
- Builds trust and credibility

### 📱 Responsive Design
- Fully responsive across mobile, tablet, and desktop
- Mobile-first approach using Bootstrap
